# TP Livre d'Or PHP

Petit système de livre d'or stockant les données dans un fichier texte.

## Fonctionnalités
- Enregistrement des avis dans `livre_d_or.txt`.
- Format de stockage : `<<Nom|Email|Date|Message>>`.
- Affichage limité aux **5 derniers messages** postés.

## Installation
1. Copiez les fichiers dans votre dossier `htdocs` (XAMPP) ou `www` (WampServer).
2. Assurez-vous que le serveur PHP a les droits d'écriture dans le dossier pour créer le fichier `.txt`.
3. Lancez `index.php` dans votre navigateur.
